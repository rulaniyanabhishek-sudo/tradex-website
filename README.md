# TradeX Terminal

TradeX Terminal is a premium, high-fidelity responsive trading command center dashboard.

Features:
- WebGL & Three.js 3D crystal core visualization.
- Canvas-based candlestick charts with volume, EMA overlays, and crosshairs.
- Sector rotation matrix and setups watchlist.
- Options Chain Greeks table and order transaction simulation.
- Performance Journal with equity growth charts and historical logs.
